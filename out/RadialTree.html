Class: RadialTree

RadialTree(csvFile, enableTransitions, transitionMultiplier, textSize, textDistanceFromNode, nodeOpacity, nodeColour, nodeColourOnHover, nodeRadius, branchOpacity, branchColour, branchColourOnHover, branchThickness, treeOpacityOnClick, dataBoxID, rMultiplier)

A class representing a d3 data visualization in the form of a "Radial Tidy Tree". In order to create this, an instance must be created (var x = new RadialTree([params])) then generateTree() must be run (x.generateTree()).

new RadialTree(csvFile, enableTransitions, transitionMultiplier, textSize, textDistanceFromNode, nodeOpacity, nodeColour, nodeColourOnHover, nodeRadius, branchOpacity, branchColour, branchColourOnHover, branchThickness, treeOpacityOnClick, dataBoxID, rMultiplier)

textSize string 4px the size of the text

nodeOpacity number 1 the node shape and text opacity
nodeColour string #173C82 the node shape and text colour
nodeColourOnHover string red the node shape and text colour on hover
nodeRadius number 1.3px the node radius
branchOpacity number 0.5 the branch opacity
branchColour string #5F92F2 the branch colour
branchColourOnHover string red -the branch colour on hover
branchThickness number 1px the branch thickness
treeOpacityOnClick number 0.2 the rest of the tree opacity when a node shape or text is clicked
dataBoxID string node-data the HTML ID of the place data should be placed
rMultiplier number 1 changes the distance between rings of nodes
